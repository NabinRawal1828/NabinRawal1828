## 🎲 Probability & Statistics in Practice

> *"In God we trust; all others must bring data."* — W. Edwards Deming

Here's how I apply probability & statistics in real data science work:

| Concept | Practical Use |
|---|---|
| 📐 **Descriptive Stats** | Mean, median, std, IQR — summarizing datasets with Pandas & NumPy |
| 🔔 **Distributions** | Normal, Binomial, Poisson — modelling real-world data patterns with SciPy |
| 🧪 **Hypothesis Testing** | t-tests, chi-square, p-values — validating assumptions in experiments |
| 📉 **Regression Analysis** | Linear & logistic regression — predicting outcomes with Scikit-learn |
| 🎯 **Bayesian Thinking** | Updating beliefs with new evidence — probabilistic reasoning |
| 📊 **Visualization** | Distributions, box plots, heatmaps — storytelling with Matplotlib & Seaborn |

```python
# Quick example: testing if two groups are statistically different
from scipy import stats
import numpy as np

group_a = np.array([23, 25, 28, 22, 26])
group_b = np.array([30, 32, 28, 35, 31])

t_stat, p_value = stats.ttest_ind(group_a, group_b)
print(f"p-value: {p_value:.4f} → {'Significant difference! 🎯' if p_value < 0.05 else 'No significant difference'}")
```
